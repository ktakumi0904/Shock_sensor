# 衝撃波センサ実行後の VTU ファイルと ParaView 操作ガイド

このドキュメントは、`ducros_sensor.py` および `lovely_sensor.py` / `lovely_sensor_filtered.py` を
実行して得られる VTU ファイルについて、
**(A) ファイルにどのような変化が加えられるか** と
**(B) ParaView でどう操作すれば何が確認できるか** をまとめたものです。

- **第1〜3部**: Ducros センサー (`*_ducros.vtu`)
- **第4部**: Lovely-Haimes センサー (`*_lovely.vtu` / `*_lovely_filtered.vtu`)

---

## 第1部: センサ実行後に VTU ファイルに加わる変化

### 1.1 全体像

センサは入力ファイル (例: `flow.vtu`) を **読み込むだけで上書きはせず** 、
新しいファイル `flow_ducros.vtu` を生成します。
このとき、**メッシュ形状 (節点座標・セル接続) は一切変更されません** 。
変わるのは「節点データ (point data) に 4 つのスカラー場が追加される」点だけです。

```
flow.vtu  (入力)                    flow_ducros.vtu  (出力)
├─ 節点座標         ────────────→   ├─ 節点座標         (変化なし)
├─ セル接続         ────────────→   ├─ セル接続         (変化なし)
└─ point_data                       └─ point_data
   └─ Velocity 等  ────────────→      ├─ Velocity 等   (変化なし)
                                       ├─ ducros_phi    ★追加
                                       ├─ divergence    ★追加
                                       ├─ vorticity_z   ★追加
                                       └─ shock_mask    ★追加
```

### 1.2 追加される 4 つのフィールド

| フィールド名   | 型      | 範囲     | 物理的意味                          | 主な用途                  |
| ------------- | ------- | -------- | ----------------------------------- | ------------------------- |
| `ducros_phi`  | float   | 0 〜 1   | Ducros センサー値                   | 衝撃波の連続的な強度マップ |
| `divergence`  | float   | 任意     | 速度の発散 ∇·u                      | 圧縮/膨張の確認 (検証用)  |
| `vorticity_z` | float   | 任意     | 渦度 ω_z                            | 渦領域の確認 (検証用)     |
| `shock_mask`  | int     | 0 or 1   | 衝撃波ラベル (phi ≥ threshold)      | 二値の衝撃波領域          |

## 第2部: ParaView 操作ガイド (Ducros センサー)

### 2.1 ファイルを開く

1. ParaView を起動
2. **File → Open** で `flow_ducros.vtu` を選択
3. 左下の **Properties** パネルで **`Apply`** ボタン (緑) を押す

### 2.2 基本: 色付け対象を切り替える

画面上部のツールバーに **色付け対象のプルダウン** があります
(初期値は "Solid Color")。
ここでフィールドを選ぶと、その値で色付け表示されます。

---

### 2.3 操作パターン別ガイド

#### パターン A: 衝撃波の強度マップを見る (連続値)

| 操作 | 内容 |
| ---- | ---- |
| 1 | 色付けプルダウンで **`ducros_phi`** を選択 |
| 2 | **`Edit Color Map`** を開く |
| 3 | **`Rescale to Custom Range`** で範囲を **0〜1** に固定 |
| 4 | カラーマップを `Viridis` や `Black-Body Radiation` に変更 (任意) |

#### パターン B: 衝撃波ラベルだけを抽出する (二値)

| 操作 | 内容 |
| ---- | ---- |
| 1 | **Filters → Common → Threshold** を選択 |
| 2 | `Scalars` に **`shock_mask`** を指定 |
| 3 | 範囲を **下限 1・上限 1** に設定 |
| 4 | **`Apply`** を押す |

#### パターン C: 閾値を変えながら衝撃波領域を探る

| 操作 | 内容 |
| ---- | ---- |
| 1 | **Filters → Common → Threshold** → `Scalars` に **`ducros_phi`** を指定 |
| 2 | 下限スライダーを動かして 0.8 / 0.9 / 0.95 等を試す |

#### パターン D: センサの判定根拠を検証する

`divergence` と `vorticity_z` をそれぞれ表示して比較する。
衝撃波位置で `divergence` が強く負、後流・境界層で `vorticity_z` が大きければ正常。

## 第4部: Lovely-Haimes センサー

### 4.1 出力フィールドと設計思想

Lovely-Haimes 法の出力は **衝撃波断面スカラー場 `shock_mach`** です。

```
flow.vtu  (入力)                    flow_lovely_filtered.vtu  (出力)
├─ 節点座標         ────────────→   ├─ 節点座標         (変化なし)
├─ セル接続         ────────────→   ├─ セル接続         (変化なし)
└─ point_data                       └─ point_data
   └─ Velocity 等  ────────────→      ├─ Velocity 等   (変化なし)
                                       ├─ shock_mach    ★主出力
                                       ├─ grad_p_mag    ★診断用
                                       ├─ sound_speed   ★診断用
                                       ├─ v_dot_grad_vmag ★診断用
                                       ├─ filter1_pass  ★診断用
                                       └─ filter2_pass  ★診断用
```

**`shock_mach` の定義:**

```
shock_mach[i] = M_n[i]   … 衝撃波検知条件を満たす節点
shock_mach[i] = 0         … それ以外
```

SU2 が出力する生の M_n 場から Mn=1 等値線を引くと膨張波や数値誤差の偽検知が混入しますが、
`shock_mach=1` の等値線はフィルタリング済みの **真の衝撃波断面のみ** を示します。

| フィールド名      | 主な用途                                                      |
| ---------------- | ------------------------------------------------------------- |
| `shock_mach`     | **主出力**。等値線 `=1` で衝撃波断面、カラー表示で衝撃波領域 |
| `grad_p_mag`     | Filter 1 の根拠確認、圧力勾配の強さの診断                    |
| `sound_speed`    | 音速分布の確認                                                |
| `v_dot_grad_vmag`| Filter 2 の判定値 (負 = 速度減速 = 衝撃波後方)               |
| `filter1_pass`   | Filter 1 が通過した節点の確認                                 |
| `filter2_pass`   | Filter 2 が通過した節点の確認                                 |

---

### 4.2 ParaView 操作パターン

#### パターン F: 衝撃波断面を等値線で表示 (推奨)

| 手順 | 操作 |
| ---- | ---- |
| 1 | `*_lovely_filtered.vtu` を **File → Open** で開き、`Apply` |
| 2 | **Filters → Common → Contour** を選択 |
| 3 | **Contour By** のプルダウンで **`shock_mach`** を選択 |
| 4 | **Isosurfaces** の値欄に **`1.0`** を入力 |
| 5 | `Apply` を押す → 衝撃波断面が等値線として表示される |

**確認できること**: フィルタリングを経た衝撃波断面の位置と形状。
Wu et al. (2013) Fig. 21 の衝撃波断面表示がこれに対応する。

> **Tip**: 元の流れ場 (`flow.vtu`) を背景表示し、`Pressure` で色付けしておくと、
> 圧力上昇と等値線の対応がひと目で確認できる。

#### パターン G: 衝撃波領域をカラーマップで確認

| 手順 | 操作 |
| ---- | ---- |
| 1 | 色付けプルダウンで **`shock_mach`** を選択 |
| 2 | **Edit Color Map** → `Rescale to Custom Range` で範囲を **0 〜 1.5** に設定 |
| 3 | カラーマップを **`Cool to Warm`** (青=0, 白=1, 赤=1超) に変更 |

**確認できること**: 衝撃波領域の空間的な広がりと強度分布。
Wu et al. (2013) Fig. 23 のような衝撃波領域の可視化に相当する。

#### パターン H: フィルタなし vs フィルタあり の比較

| 手順 | 操作 |
| ---- | ---- |
| 1 | `*_lovely.vtu` と `*_lovely_filtered.vtu` の両方を開く |
| 2 | 画面を Split Horizontal で 2 分割 |
| 3 | 各ビューにそれぞれのファイルを割り当て |
| 4 | 両方に Contour (`shock_mach=1`) を適用する |
| 5 | 並べて比較する |

**確認できること**: Filter 1/2 による偽検知の除去効果。
フィルタあり版では数値誤差・膨張波由来の等値線が除去されている。

#### パターン I: フィルタの効果を段階的に確認

| 手順 | 操作 |
| ---- | ---- |
| 1 | `*_lovely_filtered.vtu` を開く |
| 2 | 色付けプルダウンで `filter1_pass` を選択 → Filter 1 通過節点を確認 |
| 3 | 同様に `filter2_pass` を確認 |
| 4 | `v_dot_grad_vmag` を表示 → 負の領域 (速度減速域) が衝撃波後方に対応しているか確認 |

**確認できること**: 各フィルタがどの領域を除去しているか。
`filter1_pass=1` かつ `filter2_pass=1` の節点だけが `shock_mach > 0` になる。

---

### 4.3 Ducros センサーとの比較

| 観点             | Ducros センサー (`ducros_phi`)              | Lovely-Haimes センサー (`shock_mach`)         |
| ---------------- | ------------------------------------------- | --------------------------------------------- |
| 等値線の引き方   | `ducros_phi = 0.95` 等 (閾値は経験的)       | **`shock_mach = 1.0`** (物理的に明確な基準)   |
| 膨張波の扱い     | 分母の渦度で自動抑制                        | 圧縮フィルタ / Filter 2 で除外                |
| 偽検知の除去     | 渦度が大きいと `ducros_phi` が下がり抑制    | Filter 1 で数値誤差域を除去                   |
| 推奨用途         | 衝撃波の強度マップ・閾値感度の確認          | **衝撃波断面の正確な等値線可視化**            |

---

### 4.4 トラブル時の確認 (Lovely-Haimes 固有)

| 症状 | 確認・対処 |
| ---- | ---------- |
| `shock_mach` が全て 0 | `mn_threshold` を `0.95` に下げる / フィルタを無効にして確認 |
| Contour に何も表示されない | `shock_mach` の最大値を確認 (M=1 以下の流れでは等値線が存在しない) |
| 等値線が翼全体に広がる | `filter1_eps` を大きくする (例: `0.01`) / `grad_p_mag` を確認 |
| 衝撃波断面に欠けがある | `filter1_eps` を小さくする (例: `0.0001`) / `filter2_enabled: false` を試す |
| 等値線が複数本出る | 数値拡散で衝撃波が鈍っている → 正常。最も外側の線が真の衝撃波位置に近い |
