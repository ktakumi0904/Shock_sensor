## 3. Lovely-Haimes センサー `lovely_sensor` (177–191 行)

このスクリプトの中核。法線マッハ数を計算する。

- 177: 関数定義。引数に圧力 `p`・密度 `rho`・比熱比 `gamma` が入る。
- 182: **圧力勾配 ∇p を計算** 。`dpdx`, `dpdy`。
- 184: 圧力勾配の大きさ `|∇p| = sqrt(dpdx^2+dpdy^2)`。
- 185: **速度と圧力勾配の内積 `u·∇p`** 。これが衝撃波法線方向の流速成分に比例する。正なら圧縮、負なら膨張。
- 187: **局所音速 `a = sqrt(gamma p/rho)`** 。負値や 0 割を防ぐため `np.maximum` で下限を設ける。
- 190: **法線マッハ数 `M_n = (u·∇p)/(a|∇p|)`** 。分母に `eps` を足してゼロ割回避。
- 191: M_n・内積・勾配大きさ・音速を返す。

### M_n の意味

`u·∇p / |∇p|` は速度ベクトルを圧力勾配 (=衝撃波法線) 方向に射影した成分。これを音速 `a` で割るのでマッハ数になる。`M_n>=1` は「法線方向の流れが超音速」を意味し、衝撃波の存在を示す。

## Ducros 版との対応関係まとめ

| 項目             | Ducros 版                    | Lovely 版                          |
| --------------- | ---------------------------- | ---------------------------------- |
| 設定ファイル     | `config.yaml`                | `config_lovely.yaml`               |
| 必要フィールド   | 速度のみ                     | 速度・圧力・密度                   |
| 中核の量         | phi (発散/渦度比)            | M_n (法線マッハ数)                 |

| 出力サフィックス | `_ducros.vtu`                | `_lovely.vtu`                      |

`nodal_gradients_lsq` は両者で完全に同一。`load_su2_vtu` と中核センサー関数が主な差分箇所。
